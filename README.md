# ScamCheck

AI destekli telefon numarası risk analizi uygulaması.

## Teknolojiler

- Flutter
- FastAPI
- SQLAlchemy
- SQLite
- OpenAI API

## Özellikler

- Telefon numarası ekleme
- Şikayet sistemi (geliştiriliyor)
- AI analiz (yakında)

## Kurulum

```bash
pip install -r backend/requirements.txt
```

Sunucuyu çalıştır:

```bash
python -m uvicorn app.main:app --reload
```
